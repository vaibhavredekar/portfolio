# VR Portfolio – CV & Project Showcase

Personal developer portfolio website showcasing projects, skills, and experience with live theme switching (dark/light).

## ✨ Features
- Responsive layout with project cards
- Dark / Light theme toggle (persists via `data-theme`)
- 6+ project highlights (ML, IoT, Geospatial, Tableau, LoRa)
- Dynamic logo that adapts to theme accent color
- Clean, modern design

## 🛠️ Tech Stack
- HTML5 / CSS3 (CSS custom properties for theming)
- Vanilla JavaScript (theme switching)
- SVG icons

## 🚀 Quick Start
1. Clone or download the repository
2. Open `index.html` in your browser
3. Click the theme toggle (sun/moon) to switch between dark and light modes

## 📁 Project Structure
├── index.html
└── assets/
└── (image)


## 🎨 Customization
- Theme colors are defined in `[data-theme="dark"]` and `[data-theme="light"]` CSS blocks
- Replace project content in the `.proj-card` divs
- Update your name, bio, and social links as needed
